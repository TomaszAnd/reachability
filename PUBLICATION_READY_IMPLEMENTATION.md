### 2. Floor-Aware Plotting

**Implemented in**: `reach/viz.py`

**Helper function**:
```python
def _create_floor_masked_array(y_values: np.ndarray, floor: float) -> np.ma.MaskedArray:
    """
    Create masked array that breaks line segments at floor values.
    Prevents vertical "cliff" lines when values hit display floor.
    """
    is_floored = np.abs(y_values - floor) < floor * 0.01
    return np.ma.masked_where(is_floored, y_values)
```

**Application** (in both plot functions):
1. Detect floored points: `is_floored = np.abs(p - floor) < floor * 0.01`
2. Plot non-floored with masked array: `p_masked = _create_floor_masked_array(p, floor)`
3. Plot floored separately: `ax.plot(k[is_floored], p[is_floored], ..., alpha=0.3, linestyle='none')`
4. Add annotation when floor is hit
