# Quantum State Reachability Analysis

## Project Overview
Analysis of phase transitions in quantum state reachability using three criteria:
- **Moment Criterion**: Based on moment constraints
- **Spectral Criterion**: Based on spectral overlap with threshold τ
- **Krylov Criterion**: Based on Krylov subspace reachability with threshold τ

## Key Results

### Scaling Laws (τ=0.99)
- **Moment**: K_c ≈ 0.40d + 0.3, ρ_c decreases with d
- **Spectral**: K_c ≈ 1.95d - 5.9, ρ_c nearly constant
- **Krylov**: K_c ≈ 0.97d - 0.2, ρ_c nearly constant

### Fit Quality (Mean R²)
- **Fermi-Dirac** (2 params): 0.958
- **Richards** (3 params): 0.973

### Tau Dependence
- **K_c increases with τ**: Higher thresholds require more operators
  - d=10 Krylov: K_c = 7.18 (τ=0.80) → 9.05 (τ=0.99)
- **Δ decreases with τ**: Sharper phase transitions at higher thresholds
  - d=10 Krylov: Δ = 0.46 (τ=0.80) → 0.09 (τ=0.99)
